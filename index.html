우리는 무엇을 놓치고 있을까요?

우리가 가진 수많은 캠페인 데이터들, 지금 어디에, 어떻게 쓰이고 있나요?
분산되고 단절된 채 사일로에 갇혀 의사결정과 실행에서 제 역할을 하지 못하고 있습니다.

우리는 이미 데이터가 충분히 있지만, 그 잠재력을 끌어내지 못하고 있습니다.

데이터 고립은 조직의 경쟁력을 저해시킵니다.

데이터가 분산되고 통합되지 않으면 여러 문제점이 발생하며, 이는 결국 기업에 실제적인 손해로 이어집니다.

사일로화된 데이터를 자산으로 전환하려면,
캠페인 데이터의 통합·표준화가 필요합니다.

Project ATLAS

All-in-one Total Linkage for Ad Systems

ATLAS는 파편화된 전사 데이터를 통합하여 유실을 방지하고, 데이터 자산의 가치를 극대화하기 위해 시작한 프로젝트입니다.

광고 데이터 통합, 왜 지금까지 못했을까요?

과거에도 데이터 통합을 위한 시도는 있었지만, 기존 시스템은 데이터를 단순히 쌓고 있는 **‘히스토리 뷰어’**가 되었습니다.

이러한 문제는 데이터 파편화/유실의 문제 해결을 위한 데이터 통합이 시스템 기획의 목표이자 목적이 되었기 때문입니다.

우리는 ‘데이터는 있지만, 활용할 수 없다’는 고질적인 문제 해결을 위해
데이터 통합을 통해 어떤 가치를 제공할 수 있을지 고민하고
실제 사용자들이 원하는 것들은 무엇인지 의견 수렴을 통해 광고 데이터를 온톨로지화 했습니다.

이를 통해 진정한 광고 데이터 통합 시스템을 기획했습니다.

Before

  • 문제 데이터 파편화, 유실
  • 목표 데이터 통합
  • 목적 데이터 축적
  • 과정 매체별 데이터 제공 항목 확인 후, 표준화 가능한 항목으로 데이터 통합
  • 결과 파편화된 데이터의 나열, 혹시 모르니 찾아보는 단순 DB

After (ATLAS)

  • 문제 데이터 파편화, 유실
  • 목표 데이터 가치 극대화
  • 목적 데이터 활용 방안 계획
  • 과정 인터뷰를 통한 실사용자 니즈 확인, 광고 데이터 온톨로지 설계 후 표준화/통합
  • 결과 유기적으로 연결된 정보와 지식, 데이터에 기반한 의사결정 도구

ATLAS 캠페인 데이터 통합 과정

ATLAS는 온톨로지 기반으로 데이터를 구조화하고, 매체별로 상이하게 수집되던 데이터를 표준화·정규화하여 일관성, 신뢰도를 확보했습니다. 또한, 데이터 라벨링을 통해 활용성​을 크게 향상시켰습니다.

데이터 구조화

(온톨로지)

데이터 표준화

데이터 정규화

데이터 라벨링

1. 온톨로지 기반으로 데이터를 구조화

2. 데이터 표준화

데이터 표준화를 위해, 매체에서 제공되는 데이터 항목을 세부적으로 파악합니다.
데이터 항목은 무엇이며, 해당 데이터 항목을 어느 수준까지 제공하는지 세부 확인합니다. 필요한 항목이 대시보드/리포트에서 확인되지 않는 경우, API로 제공되는 항목인지 확인합니다.

3. 데이터 정규화 & 4. 데이터 라벨링

데이터 정규화

데이터 정규화를 위해, 매체에서 제공되는 분석기준 항목의 데이터 구분값을 세부적으로 파악합니다.

데이터 라벨링 (AI 자동화)

데이터 라벨링을 기존의 수작업 방식에서 AI기술을 도입한 자동화 방식으로 전환합니다.

광고 대행업의 경쟁력,
업무 효율화가 기업 생존과 직결될 것

경기 침체로 인한 광고예산 축소

고금리, 고물가, 경기 둔화가 지속되며 기업 마케팅예산 삭감

과도한 경쟁으로 수익성 악화

제한된 물량 → 출혈 경쟁(최저가 입찰)

고질적인 인력 문제

우수 인력 이탈 → 서비스 품질·조직 안정성 하락 → 악순환

가장 먼저 무엇을 만들면 좋을까요?

Digital First · Digital Trinity 내 9개 팀, 총 21명과의 대면 인터뷰 결과,
가장 시급한 Pain Point는 **‘레퍼런스 부족으로 인한 매체 운영 및 광고주 대응의 어려움’** 으로 나타났습니다.

현업 중심의 두 가지 솔루션을 기획했습니다

최고의 성과를 낸 레퍼런스 데이터를 기반으로 자동화를 구현해 효율성과 정확성을 높입니다.

미디어 믹스 자동화 시스템 (가제)

소재 성과 및 레퍼런스 기반 추천 시스템

이 시스템, 얼마나 효율적이고 수익성이 있을까요?

시간적 리소스를 N% 감소시킬 수 있을 것으로 기대하며,
이는 마진율 7% 기준, N원 절감 → N원 매출 증가 효과와 동일합니다.

지속적인 고도화와 프로덕트 확장을 통해 더 큰 가치를 만들어갑니다.

모니터링·예측·최적화까지 가능한 **디지털 트윈** 환경 구축으로 확장됩니다.
