광고 데이터 통합, 왜 지금까지 못했을까?

1. 단순한 취합이 아니기 때문입니다.
흩어진 데이터를 정리하고 분류·선별하는 복잡한 구조화 작업입니다.

사용자 니즈에 기반한 구조화가 없었기에, 데이터가 있어도 제대로 활용되지 못했습니다.

Page 9

Project ATLAS

All-in-one Total Linkage for Ad Systems

ATLAS는 파편화되고 유실되는 전사 데이터를 통합하여
데이터 자산의 가치를 극대화하기 위해 시작한 프로젝트입니다.

Page 10

ATLAS 캠페인 데이터 통합 과정

ATLAS는 온톨로지 기반으로 데이터를 구조화하고, 매체별로 상이하게 수집되던 데이터를
표준화·정규화하여 일관성, 신뢰도를 확보했습니다.
또한, 데이터 라벨링을 통해 활용성​을 크게 향상시켰습니다.

1. 온톨로지 기반으로 데이터를 구조화

Page 12

2. 데이터 표준화

데이터 표준화를 위해, 매체에서 제공되는 데이터 항목을 세부적으로 파악합니다.
데이터 항목은 무엇이며, 해당 데이터 항목을 어느 수준까지 제공하는지 세부 확인합니다. 필요한 항목이 대시보드/리포트에서 확인되지 않는 경우, API로 제공되는 항목인지 확인합니다.

Page 13

3. 데이터 정규화 & 4. 데이터 라벨링

데이터 정규화

데이터 정규화를 위해, 매체에서 제공되는 분석기준 항목의 데이터 구분값을 세부적으로 파악합니다.

데이터 라벨링 (AI 자동화)

데이터 라벨링을 기존의 수작업 방식에서 AI기술을 도입한 자동화 방식으로 전환합니다.

Page 14

ATLAS, 우리 데이터의 미래입니다.

ATLAS는 단순한 데이터 통합 DB를 넘어, 자동화 기반의 업무 혁신을 이끌며,
모니터링·예측·최적화까지 가능한 **디지털 트윈** 환경 구축으로 확장됩니다.
